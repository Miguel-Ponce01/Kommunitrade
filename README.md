# KomuniTrade

KomuniTrade is a hyper-local, community-driven marketplace web application designed specifically for Philippine barangays. It simplifies the process of buying and selling within immediate neighborhoods, ensuring trust, privacy, and convenience. 

This repository contains the **Phase 1 MVP Prototype**, which focuses on a premium, mobile-first user interface and the core "happy path" user flow using mock data.

## 🌟 Features (MVP Prototype)
* **Mobile-First Design**: A clean, app-like interface optimized for mobile browsers, ensuring accessibility for all users.
* **Hyperlocal Filtering**: Browse listings specific to your barangay (e.g., Obrero, Agdao) instantly via dropdown filtering.
* **Frictionless Posting**: A streamlined "Sell an Item" interface designed to be fast and intuitive.
* **Modern UI**: Built with a vibrant, trustworthy color palette, subtle glassmorphism effects, and smooth micro-animations.

## 🛠️ Technology Stack
* **Frontend Framework**: React
* **Build Tool**: Vite
* **Routing**: React Router
* **Styling**: Vanilla CSS (Custom Variables, Mobile-First)
* **Icons**: Lucide React

## 🚀 Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation
1. Clone the repository or navigate to the project folder:
   ```bash
   cd KomuniTrade
   ```
2. Install the dependencies:
   ```bash
   npm install
   ```

### Running the App Locally
Start the Vite development server:
```bash
npm run dev
```
Open your browser and navigate to the local URL provided in the terminal (usually `http://localhost:5173`).
